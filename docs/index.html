PostgreSQL 的版本

PostgreSQL9.2 开始支持 json 类型，但函数不全，不建议使用，9.5 版本加入了 jsonb_set 函数，9.6 版本加入了 jsonb_insert 函数，变更 json 内容变得非常容易，建议使用 9.5 以后的版本。
